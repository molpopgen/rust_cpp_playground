# Experiments with rust/cpp interaction

## Getting started

```
gh repo clone molpopgen/rust_cpp_playground
cd rust_cpp_playground
git submodule update --init
```

## Building

### Requirements

* [rust](https://www.rust-lang.org/tools/install)
* A c++ compiler.

### GNU make

```
make
```

### cmake

```
mkdir build
cmake ..
make
```
